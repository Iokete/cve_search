## cve_search
Proyecto para la asignatura auditoría, que presenta un script en Python que realiza búsquedas de CVE y CPE mediante el uso de las APIs de NVE: https://nvd.nist.gov/developers/vulnerabilities https://services.nvd.nist.gov/rest/json/cves/2.0) y CPE: https://nvd.nist.gov/developers/products
